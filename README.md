# eslint-config-simple

<a name="install"></a>

### Install

First install `eslint-config-simple`:

```shell
$ yarn install eslint-config-simple --dev
```

Then add to the `extends` array in your eslint config file:

```json5
{
  extends: ["simple"]
}
```
